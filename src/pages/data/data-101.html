{{#markdown }} ## How the Orbit Carousel is set up The images are added as front matter at the top of page on which they are displayed. Which is rather convenient. ``` --- aria-label: Images of china image1: ../assets/img/orbit/china/china1.jpg caption1: Image of China 1. image2: ../assets/img/orbit/china/china2.jpg caption2: Image of China 2! image3: ../assets/img/orbit/china/china3.jpg caption3: Image of China 3!! image4: ../assets/img/orbit/china/china4.jpg caption4: Image of China 5!!! --- ``` The carousel is inserted on the page using a panini tag. ```html \{{> orbit-image-carousel}} ``` The tag refers to the filename that acts as a template to format the data. This template calls for 4 sets of data, one per iamge. This data is just what we provide in the front matter @ partials/orbit-carousels/orbit-image-carousel.html ```html
``` ## In General Custom data can be added to your pages via Front Matter. This data can then be inserted into your HTML through Panini Tags. To add variables to a **specific page** only, add it at the top of the page's HTML as a front-matter block. Let's say the below content is @ src/pages/index.html. ```html --- title: Page Title description: Lorem ipsum. --- ``` With the front matter listed on the page you can insert the values of these variables into the page using the tags `\{{title}}` and `\{{description}}`. ```html

\{{ title }}
