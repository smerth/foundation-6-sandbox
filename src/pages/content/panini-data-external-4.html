{{#markdown }} # Panini Data: External JSON #4 Open up your browser's console. Check the log. You will see the json response object from Github containing the users repo data. Documentation for obtaining a jsonp response from Github is found here: [Github API v3.0](https://developer.github.com/v3/) Here is the handler function that is used to print the jsonp data to the console. ```javascript function foo(response) { var meta = response.meta; var data = response.data; console.log(meta); console.log(data); } var script = document.createElement('script'); script.src = 'https://api.github.com/users/smerth/repos?callback=foo'; document.getElementsByTagName('head')[0].appendChild(script); ``` The query string ```?callback=foo``` appended to the request url tells the api to wrap the resonse json object in a callback function with the arbitrary name ```foo``` Then you need to write the above function to handle ```foo```. {{/markdown }}
