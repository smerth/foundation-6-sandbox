Panini Data: Local Files

A local json or yaml file can be parsed by Panini into variables which can be used in templates.
{{#markdown }} ### Data The datafile is placed in the data folder (in this case ```hsk-data.json```) It has the following structure. ```json { "hsk-all-words": [ { "word": { "title": "\u505a\u4e3b", "field_definition": "make the decision; take charge of; back up; support; be host", "field_word_pronunciation": "zuo4 zhu3" } }, { "word": { "title": "\u5ea7\u53f3\u94ed", "field_definition": "motto; maxim", "field_word_pronunciation": "zuo4 you4 ming2" } } ] } ``` ### Template The data is accessed through variables made available by Panini auto-magically. To drilldown into the data you use dot notation. ```filename.array``` gets you into the data set. ```this.word.title``` gets you to the data-item you need. ```html
    \{{#each hsk-data.hsk-all-words}}
  1. \{{ this.word.title }}
    2. \{{/each}}
``` {{/markdown }}
