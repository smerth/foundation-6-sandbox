Photoswipe

Since "Clearing" is removed from Foundation 6, you need to find another lightbox solution. Photoswipe is one option. The main point of using Photoswipe is to get swiping action going. If that doesn't matter then Magnific might be a better option.

Notes.

Photoswipe should be installed using Bower since the other assets in this build are using it. Skin assets were manually pasted into the css folder so the paths from the css file will work, this is an issue of processing the scss... When using images from Flickr you need to pay attention to the URLs for each size image. Gallery is intiated using the class "demo-gallery." Try multiple galleries per page...
